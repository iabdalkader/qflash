<h1 align="center">QFlash</h1>

<p align="center">
  A simple tool for managing LittleFS flash storage on Arduino Uno-Q via ADB and OpenOCD.
</p>

## Building

Requires Node.js and Rust.

```
npm install
npm run tauri dev
```
